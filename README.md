This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app) developed as a submission requirement for videodubber.ai internship.

## Getting Started

### Clone the repo

You can clone the repo by using

```bash
git clone https://github.com/rayaanoidPrime/videodubber-assignment
```

### Installing node_modules

```bash
cd videodubber-assignment
npm i
```

### Running dev server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
